****************************************************************

		ティラノスクリプト用テーマ一括変換プラグイン

****************************************************************

　ダウンロードいただき誠にありがとうございます。ゲーム制作の一助となれば幸いです
　テーマ一括変換プラグインはティラノスクリプト v440 以降のバージョンで使用することができます。
　動作確認は v471 で行っています。

――――――――――――――――――――――――――――――――
■ファイル詳細
――――――――――――――――――――――――――――――――
　theme_kopanda_03 は以下の２種類を同梱しております。
　制作されるゲームの画面サイズに合わせてお選びください。

　　theme_kopanda_03a：幅640px、高さ960px（縦長）
　　theme_kopanda_03b：幅960px、高さ640px（横長）


　material フォルダにはカスタム用の素材が入っています。
　おまけモード（CG閲覧、シーン回想、BGM視聴など）を作成したり
　ボタンに表示するテキストを変更したいときなどにどうぞ。


　nextpage.gif は thme_kopanda_03 に合わせたクリック待ちグリフです。
　tyrano > images > kag 内のクリック待ちグリフに上書きしてご利用ください。

――――――――――――――――――――――――――――――――
■機　能
――――――――――――――――――――――――――――――――
　プラグインを使用することでゲームのシステムテーマを簡単に変更することができます。

　変更できるテーマ

　・メニュー画面
　・コンフィグ画面
　・セーブ画面
　・ロード画面
　・バックログ画面
　・メッセージウィンドウ
　・ロールボタン
　・glink タグの color パラメータに ts03a（または ts03b）を追加


――――――――――――――――――――――――――――――――
■つかいかた
――――――――――――――――――――――――――――――――
--------------------------------------------------------------------------
①プラグインの配置
--------------------------------------------------------------------------
　プロジェクトフォルダの data/others/plugin に

　画面サイズ 640×960px なら「theme_kopanda_03a」フォルダ
　画面サイズ 960×640px なら「theme_kopanda_03b」フォルダ

　を配置します。

--------------------------------------------------------------------------
②画面サイズの変更（※画面サイズ 640×960px の場合のみ）
--------------------------------------------------------------------------
　プロジェクトフォルダの data/system/config.tjs を開き
　画面サイズを width=640, height=960 に変更しておきます

--------------------------------------------------------------------------
③プラグインの導入
--------------------------------------------------------------------------
　画面サイズ 640×960px なら　[ plugin name="theme_kopanda_03a" ]
　画面サイズ 960×640px なら　[ plugin name="theme_kopanda_03b" ]

　上記のタグを記述するとプラグインを読み込み、即時、テーマが反映されます。
　タグの挿入位置はゲーム全体で反映させる場合は first.ks などが良いでしょう。

（注）first.ksでプラグインを読み込んだあとに [ position ] などでメッセージ枠の設定を上書きしないようご注意ください。

--------------------------------------------------------------------------
④機能ボタン（ロールボタン）の表示
--------------------------------------------------------------------------
　機能ボタンを表示させるには [ add_theme_button ] と記述します。
　非表示にしたい場面では [ clearfix ] タグを記述します。

　機能ボタンの表示位置、ボタンのサイズ、不要なボタンの削除などは
　プラグインフォルダ内の init.ks を編集することでカスタマイズができます。

--------------------------------------------------------------------------
⑤引　数
--------------------------------------------------------------------------
　プラグインのロード時に引数を渡すこともできます。詳細は init.ks をごらんください。

例）
　[plugin name="theme_kopanda_03" font_color="0xFFFFFF" name_color="0xFFCC00"  frame_opacity="255" ]

　font_color: メッセージの文字色を指定できます。
　name_color:名前欄の色を指定できます。
　frame_opacity:メッセージ枠の透明度を0〜255の範囲で指定して下さい。0が完全に透明

--------------------------------------------------------------------------
⑥ginkの使用
--------------------------------------------------------------------------
　テーマに合ったglinkも自動的に使用できるようになります。
　glink タグの color に ts03 を指定して下さい。

　画面サイズ 640×960px なら color=ts03a と書きます
　画面サイズ 960×640px なら color=ts03b と書きます

例）
　[glink  color="ts03a"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="100"  text="はい。興味あります"  target="*selectinterest"  ]
　[glink  color="ts03a"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="170"  text="興味あります！"  target="*selectinterest"  ]
　[glink  color="ts03a"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="240"  text="どちらかと言うと興味あり"  target="*selectinterest"  ]
　[s]

　glinkのデザインは各プラグインフォルダ内の ts03a.css, ts03b.cssで設定しています。
　お好みに合わせてカスタマイズしてやってください。

--------------------------------------------------------------------------
⑦コンフィグ画面
--------------------------------------------------------------------------
　テーマに合ったコンフィグ画面を利用するためには
　[ button ] タグで role=sleepgame を指定してジャンプさせます。
　次のようなボタンでコンフィグ画面に移動して下さい。

例）
　[button x=166 y=507 graphic="title/button_config.gif" storage="../others/plugin/theme_kopanda_03/config.ks" role="sleepgame"]

　※storageがプラグイン以下の config.ks になっていること

■差し替え用素材について
――――――――――――――――――――――――――――――――
